она читается как размытая. Поэтому для 5 аватарок сделаны отдельные файлы:

## Требования
- Формат: PNG (для сундука/маски — желательно с прозрачным фоном).
- Кота, луну-зал, медальон можно с чёрным фоном — они показываются в тёмных рамках.
- Рекомендуемый размер: длинная сторона 1024–1536 px.

## Как добавить новый ассет
Открой `index.html`, найди объект `ASSETS` в `<script>` — добавь туда ключ и путь.
Слоты с `<img class="art-asset">` сами появятся поверх плейсхолдера.
